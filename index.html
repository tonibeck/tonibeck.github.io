Unifi Network Controller on Azure

2020-08-18 · 674 words · 4 minute read
azure · networking · unifi · infrastructure · wireless

Objectives In this post I’ll show how to deploy a Unifi Network Controller on Microsoft Azure Cloud. The main goal of this effort was to have a lightweight, efficient but yet very capable home network. In order to do so, the number of boxes required on-prem is kept to the bare minimum. The only devices that remained on-prem are a compact PoE Router (Unifi Edge Router PoE) as well as three Unifi Access Points (two pros and one light).
